# Teams考勤数据 Skill

从360teams平台获取员工考勤数据，计算有效工作日和平均工时。

## 前置配置

在项目根目录 `.env` 文件中配置：

```env
TEAMS_EM_CODE=你的员工编码
TEAMS_AUTHORIZATION=你的授权令牌
```

## 功能

1. **考勤数据获取** - 从360teams API获取指定月份考勤明细
2. **工时统计** - 自动计算已工作天数、已累计工时、当前平均工时、剩余工作日
3. **达标预测** - 基于目标工时（10/10.5/11小时）计算剩余工作日每日所需工时
4. **CLI支持** - 命令行直接查询
5. **CSV导出** - 自动保存考勤明细到CSV文件（默认 `data/attendance_YYYYMM.csv`）

## 使用方法

### 命令行调用

```bash
# 查询当前月份
python .claude/skills/teams-attendance/scripts/fetch_attendance.py

# 查询指定月份
python .claude/skills/teams-attendance/scripts/fetch_attendance.py --month 2026-01

# 导出CSV到指定路径
python .claude/skills/teams-attendance/scripts/fetch_attendance.py --month 2026-01 --output ./data/attendance_202601.csv
```

### 代码调用

```python
from scripts.fetch_attendance import parsed_att_date

# 获取2026年1月考勤统计（输出格式固定，始终打印到终端）
avg_hours, att_df = parsed_att_date("2026-01")

# 指定CSV输出路径
avg_hours, att_df = parsed_att_date("2026-01", output_path="./data/att.csv")

# 返回值可进一步处理
print(f"平均工时: {avg_hours}")
```

## 参数说明

| 参数 | 类型 | 说明 |
|-----|------|-----|
| check_month | str | 查询月份，格式 YYYY-MM |
| output_path | str | CSV输出路径，默认 `data/attendance_YYYYMM.csv` |

**注意**：输出格式固定，不可更改。统计信息始终以固定格式打印到终端。

## 输出格式

### 终端输出（固定格式）

```
═══════════════════════════════════════════════════════════
  考勤统计 - 2026-03
═══════════════════════════════════════════════════════════
  当前日期         2026-03-04
  已工作天数       2 天
  已累计工时       21.0 小时
  当前平均工时     10.5 小时/天

  剩余工作日       20 天

┌────────────┬──────────────┐
│  目标工时   │  剩余日均需  │
├────────────┼──────────────┤
│   10.0 h   │     10.0 h   │
│   10.5 h   │     10.5 h   │
│   11.0 h   │     11.1 h   │
└────────────┴──────────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  本月明细
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

│  日期   │ 上班时间 │ 下班时间 │  工时   │  备注  │
│---------│----------│----------│---------│---------│
│ 03-02   │ 09:14    │ 20:45    │ 11.52h  │        │
│ 03-03   │ 09:14    │ 18:43    │  9.47h  │        │

共 2 个工作日，累计 21.00 小时

考勤明细已保存至: data/attendance_202603.csv
```

**说明**：
- 输出格式固定，包含：基础统计、达标预测、本月明细三部分
- 已工作天数：不包含今天，仅统计工作日（isrest=0）
- 已累计工时：不包含今天的工时总和
- 剩余工作日：本月范围内，包含今天及之后的工作日
- 本月明细：仅显示已发生的工作日（排除今天和节假日）
- 若剩余工作日为 0，显示"本月已结束"

### DataFrame 格式

| 列名 | 说明 |
|-----|------|
| 日期 | 考勤日期 |
| 月份 | 所属月份 |
| 假期flag | 是否假期 (0=工作日) |
| 上班打卡 | 上班打卡时间 |
| 下班打卡 | 下班打卡时间 |
| 备注 | 考勤状态 (迟到/早退/年假等) |
| 工时 | 当日工时 (小时) |
| 有效工作日 | 0/0.5/1 |

当用户调用此命令时，请执行以下操作：

1. **确定搜索主题**：解析用户在命令后输入的参数，或询问用户想要探索的主题。
2. **推演关键词**：根据主题推演 2-3 个核心搜索词。
3. **执行抓取**：调用 `xiaohongshu-scraper` Skill 中的 `fetch_xhs.py` 脚本获取原始数据。
4. **⚠️ 强制输出流程 (必须执行)**：
   - 在 `data/xiaohongshu/` 下创建专属结果文件夹。
   - **必须**为抓取到的每一篇帖子生成一个独立的 Markdown 文件（包含内容摘要和评论区总结）。
   - **绝对禁止**将多篇帖子合并、或跳过单篇总结步骤。
   - **最后**生成汇总索引 `_index.md`，并确保包含指向所有单篇 MD 的链接。

请引导用户一步步完成搜索、登录（如果需要扫码）到最终总结产出的全过程。

# 提案：增加AlphaVantage金融市场分析能力

## 目标
扩展现有的 AlphaVantage Skill，增加基本面数据、宏观经济指标和技术指标的获取能力，以支持更全面的金融市场分析。

## 范围
- **新增**: 公司概况、财务报表（基本面）。
- **新增**: 关键经济指标（GDP、CPI、国债收益率等）。
- **新增**: 核心技术指标（SMA、RSI、MACD等）。
- **新增**: 商品数据（WTI、Brent、黄金等）。
- **排除**: 极低频或极其边缘的指标（如特定单一农业品种详细数据），除非具有宏观意义。

## 为什么
当前 `finance-data-news` 仅覆盖新闻、情绪和简单的涨跌幅排行，缺乏深度分析所需的定量数据。通过引入基本面、宏观和技术指标，Agent 将具备更完整的“金融市场分析”能力，能够从定性（新闻）和定量（指标）两个维度辅助用户。

## 变更内容
- **新增**: 公司概况、财务报表（基本面）。
- **新增**: 关键经济指标（GDP、CPI、国债收益率等）。
- **新增**: 核心技术指标（SMA、RSI、MACD等）。
- **新增**: 商品数据（WTI、Brent、黄金等）。
- **排除**: 极低频或极其边缘的指标（如特定单一农业品种详细数据），除非具有宏观意义。

## 影响
- 受影响规范：`financial-market-analytics`（新增）
- 受影响代码：`finance-data-news` 下的新增脚本
- 外部依赖：无新增，复用 Alpha Vantage API Key

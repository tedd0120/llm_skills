## 上下文

**当前状态**：
- 工时技能输出基础统计 + 达标预测 + CSV 保存提示
- DataFrame 包含完整考勤数据，可用于生成明细表

**约束**：
- 明细表必须在达标预测表格之后、CSV 保存提示之前输出
- 格式为 Markdown 表格
- 数据需过滤：本月 + 已发生（排除今天）+ 工作日（排除节假日）

## 目标 / 非目标

**目标：**
- 以 Markdown 表格形式展示本月工作日明细
- 日期/时间简化显示，工时保留 2 位小数
- 底部显示汇总信息（工作日天数、累计工时）

**非目标：**
- 不支持自定义字段选择
- 不支持排序或过滤功能

## 决策

| 决策 | 选择 | 理由 |
|-----|------|------|
| 输出格式 | Markdown 表格 | 兼容性好，易于阅读 |
| 日期格式 | MM-DD | 简洁，月份已由标题说明 |
| 时间格式 | HH:MM | 去掉秒，更简洁 |
| 工时格式 | X.XXh | 保留 2 位小数，带单位 |

## 风险 / 权衡

| 风险 | 缓解措施 |
|-----|---------|
| 数据量大时表格过长 | 本月最多 23 个工作日，表格长度可控 |
| 缺卡数据处理 | 过滤掉假期 flag，但缺卡工作日仍显示 |

## 迁移计划

- 仅修改 `print_statistics()` 函数，无需特殊迁移步骤

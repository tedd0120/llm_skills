## 为什么

当前工时技能仅输出汇总统计，用户无法快速查看本月每日的详细考勤记录。添加本月明细表可以让用户在终端直接查看每日打卡情况，无需打开 CSV 文件。

## 变更内容

- **新增**：在达标预测表格之后输出本月工作日明细表
- **新增**：明细表包含日期、上班打卡、下班打卡、工时、备注字段
- **新增**：过滤条件为本月 + 已发生（排除今天）+ 工作日（排除节假日）

## 功能 (Capabilities)

### 新增功能
- `attendance-detail-table`: 本月工作日明细表输出，以 Markdown 表格形式展示每日考勤记录

### 修改功能
- 无

## 影响

- **代码变更**：`.claude/skills/teams-attendance/scripts/fetch_attendance.py`
  - 在 `print_statistics()` 函数中新增明细表输出逻辑
- **用户体验**：终端输出增加明细信息，便于快速查看每日打卡情况
- **向后兼容**：不影响函数返回值和 CSV 导出

## 新增需求
### 需求：A股历史数据抓取
系统必须提供 A股历史数据抓取能力，使用 akshare 库的 `stock_zh_a_hist()` 函数获取指定股票的历史行情数据。

#### 场景：抓取单只A股历史数据
- **当** 用户指定股票代码（如 `000001`）、周期（daily/weekly/monthly）和日期范围
- **那么** 系统调用 akshare 抓取数据并返回包含日期、开盘价、最高价、最低价、收盘价、成交量的 DataFrame

#### 场景：数据持久化为CSV
- **当** 用户请求保存数据
- **那么** 系统将 DataFrame 保存为 CSV 文件到 `data/` 目录

### 需求：A股实时行情获取
系统必须提供 A股实时行情获取能力，使用 akshare 库的 `stock_zh_a_spot_em()` 函数获取全市场实时行情快照。

#### 场景：获取全市场A股实时行情
- **当** 用户请求实时行情
- **那么** 系统返回包含所有A股当前价格、涨跌幅、成交量等信息的 DataFrame

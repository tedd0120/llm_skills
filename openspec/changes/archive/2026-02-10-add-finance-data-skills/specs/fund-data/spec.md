## 新增需求
### 需求：基金净值数据抓取
系统必须提供基金净值数据抓取能力，使用 akshare 库获取开放式基金和 ETF 的历史净值数据。

#### 场景：抓取开放式基金净值
- **当** 用户指定基金代码（如 `000001`）
- **那么** 系统调用 akshare 的 `fund_open_fund_info_em()` 抓取净值数据，返回包含日期、单位净值、累计净值、日增长率的 DataFrame

#### 场景：抓取ETF历史数据
- **当** 用户指定 ETF 代码（如 `510300`）、周期和日期范围
- **那么** 系统调用 akshare 的 `fund_etf_hist_em()` 抓取历史行情，返回 OHLCV 格式的 DataFrame

### 需求：基金列表查询
系统必须提供基金列表查询能力，获取可用基金的代码和名称列表。

#### 场景：查询所有公募基金
- **当** 用户请求基金列表
- **那么** 系统返回包含基金代码、名称、类型的 DataFrame

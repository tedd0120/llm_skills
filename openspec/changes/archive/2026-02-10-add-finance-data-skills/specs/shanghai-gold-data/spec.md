## 新增需求
### 需求：沪金历史数据抓取
系统必须提供上海黄金交易所黄金现货历史数据抓取能力，使用 akshare 库的 `spot_hist_sge()` 函数。

#### 场景：抓取沪金Au9999历史数据
- **当** 用户请求抓取沪金数据，指定品种（如 `Au9999`、`Au9995`、`Au100g`）
- **那么** 系统调用 akshare 的 `spot_hist_sge(symbol)` 抓取历史数据，返回包含日期、开盘价、最高价、最低价、收盘价、成交量的 DataFrame

#### 场景：查询沪金可用品种
- **当** 用户请求查询可用的沪金品种
- **那么** 系统返回上海黄金交易所支持的黄金品种列表（Au9999, Au9995, Au100g, mAu 等）

### 需求：沪金实时行情获取
系统必须提供沪金实时行情获取能力。

#### 场景：获取沪金实时价格
- **当** 用户请求沪金实时行情
- **那么** 系统调用 akshare 相关接口返回当前黄金价格

## 新增需求
### 需求：美股历史数据抓取
系统必须提供美股历史数据抓取能力，使用 yfinance 库获取美国股票市场的历史行情数据。

#### 场景：抓取单只美股历史数据
- **当** 用户指定股票代码（如 `AAPL`、`GOOGL`、`MSFT`）和日期范围
- **那么** 系统调用 `yf.download(symbol, start, end)` 抓取数据，返回 OHLCV 格式的 DataFrame

#### 场景：批量抓取多只美股
- **当** 用户指定多个股票代码列表
- **那么** 系统批量下载所有股票数据，返回多层索引的 DataFrame 或分别存储

### 需求：美股公司信息获取
系统必须提供美股公司基本信息获取能力。

#### 场景：获取公司基本信息
- **当** 用户指定股票代码
- **那么** 系统调用 `yf.Ticker(symbol).info` 返回公司名称、行业、市值等信息

# 小红书报告模板规范（增量）

## 目的

为 `_index.md` 报告模板新增发散路径板块，展示 AI 在发散模式下的搜索决策路径。

---

## 新增需求

### 需求: 发散模式报告必须包含发散路径板块
当报告基于发散模式数据生成时，必须包含发散路径板块。

#### 场景: 发散路径板块渲染
- **当** `raw.json` 包含 `divergence_path` 字段且非空
- **那么** 报告必须在搜索概览之后、品牌/主题声量分析之前插入「发散路径」板块
- **并且** 发散路径板块必须使用 Markdown 表格展示每轮搜索信息

#### 场景: 发散路径表格格式
- **当** 渲染发散路径板块
- **那么** 必须使用以下表格格式：
  ```markdown
  ## 🧭 搜索发散路径

  | 轮次 | 关键词 | 实际篇数 | 发散理由 |
  |:----:|:------|:--------:|:---------|
  | 1 | [初始关键词] | N篇 | 初始搜索，建立全局认知 |
  | 2 | [关键词] | N篇 | [AI 决策理由] |
  | ... | ... | ... | ... |
  | **合计** | — | **N篇** | 去重后实际篇数 |
  ```
- **并且** 轮次列居中对齐，关键词列左对齐，实际篇数列居中对齐，发散理由列左对齐

#### 场景: 非发散模式报告
- **当** `raw.json` 不包含 `divergence_path` 字段或字段为空
- **那么** 报告必须不包含发散路径板块
- **并且** 保持现有 8 个板块结构不变

### 需求: 发散模式报告板块总数为 9 个
发散模式下报告必须包含 9 个板块（原 8 个 + 发散路径板块）。

#### 场景: 板块顺序
- **当** 生成发散模式报告
- **那么** 板块顺序必须为：
  1. 搜索概览
  2. 搜索发散路径（新增）
  3. 品牌/主题声量分析
  4. 多维对比矩阵
  5. 评论区高频共识
  6. 场景化决策建议
  7. 价格/规格参考
  8. 避雷指南
  9. 关键洞察

### 需求: 发散路径板块必须使用 emoji
发散路径板块标题必须使用 emoji 增强可读性。

#### 场景: 板块 emoji
- **当** 渲染发散路径板块标题
- **那么** 必须使用 🧭 emoji 前缀（如 `## 🧭 搜索发散路径`）

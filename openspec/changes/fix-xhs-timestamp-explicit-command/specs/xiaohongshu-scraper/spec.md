# 小红书爬虫执行规范 - 增量

此文件定义 `fix-xhs-timestamp-explicit-command` 变更的增量需求。

---

## 修改需求

### 需求:目录创建指令必须使用显式命令格式

SKILL.md 中"步骤 1：创建搜索目录"部分必须使用可执行的 Bash 命令模板，禁止使用可能诱导 Agent 模仿的具体数字示例。

#### 场景:显式命令模板要求

- **当** SKILL.md 描述目录创建步骤
- **那么** 必须提供包含 `$(date +%Y%m%d_%H%M%S)` 命令替换的完整 Bash 命令
- **并且** 禁止在示例中使用硬编码的时间数字（如 `143000`、`182300`）

#### 场景:命令执行示例格式

- **当** SKILL.md 提供目录创建示例
- **那么** 必须展示"输入命令 + 预期输出"的格式
- **示例格式**：
  ```bash
  # 输入命令
  mkdir -p "data/xiaohongshu/$(date +%Y%m%d_%H%M%S)_北京咖啡厅/"

  # 实际创建的目录（假设当前时间是 2026-03-03 21:15:30）
  # data/xiaohongshu/20260303_211530_北京咖啡厅/
  ```

#### 场景:目录路径变量保存

- **当** Agent 创建搜索目录后
- **那么** 必须将目录路径保存为变量供后续步骤使用
- **命令模板**：
  ```bash
  OUTPUT_DIR="data/xiaohongshu/$(date +%Y%m%d_%H%M%S)_[搜索主旨]/"
  ```

### 需求:目录时间戳必须使用当前系统时间

更新此需求的场景描述，补充显式命令要求。

#### 场景: 目录创建时间要求

- **当** Agent 创建搜索目录
- **那么** 必须执行 `mkdir -p "data/xiaohongshu/$(date +%Y%m%d_%H%M%S)_[搜索主旨]/"` 命令
- **并且** 禁止使用示例时间（如 `143000`）或任何虚构时间

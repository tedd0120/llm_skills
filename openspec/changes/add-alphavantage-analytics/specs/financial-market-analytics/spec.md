# 规范：金融市场分析数据

本规范定义了 Alpha Vantage 金融市场分析数据的获取能力。

## 新增需求

### 需求：获取公司基本面数据

系统**必须**支持获取公司的基本面数据，包括公司概况和主要财务报表。

#### 场景：获取公司概况 (Overview)

**假设** 用户设置了有效的 `ALPHAVANTAGE_API_KEY` 环境变量
**当** 用户请求获取 "IBM" 的公司概况
**那么** 返回的数据应包含 EPS, P/E Ratio, Revenue, Profit Margin 等关键指标

#### 场景：获取财务报表

**假设** 用户设置了有效的 `ALPHAVANTAGE_API_KEY` 环境变量
**当** 用户请求获取 "IBM" 的 "INCOME_STATEMENT" (利润表)
**那么** 返回的数据应包含 `annualReports` 和 `quarterlyReports` 列表

---

### 需求：获取宏观经济指标

系统**必须**支持获取关键宏观经济指标的时间序列数据。

#### 场景：获取实际 GDP

**假设** 用户设置了有效的 `ALPHAVANTAGE_API_KEY` 环境变量
**当** 用户请求 "REAL_GDP" 指标
**那么** 返回的数据应包含日期 (date) 和 GDP 值 (value) 的时间序列

#### 场景：获取 CPI 数据

**假设** 用户设置了有效的 `ALPHAVANTAGE_API_KEY` 环境变量
**当** 用户请求 "CPI" 指标
**那么** 返回的数据应反映最近的消费者价格指数变化

---

### 需求：获取技术分析指标

系统**必须**支持获取股票的技术指标数据。

#### 场景：获取 SMA 指标

**假设** 用户设置了有效的 `ALPHAVANTAGE_API_KEY` 环境变量
**当** 用户请求 "IBM" 的 "SMA" 指标，参数 `time_period=20`
**那么** 返回的数据应包含日期和对应的 SMA 值

---

### 需求：获取大宗商品数据

系统**必须**支持获取主要大宗商品的价格数据。

#### 场景：获取原油价格

**假设** 用户设置了有效的 `ALPHAVANTAGE_API_KEY` 环境变量
**当** 用户请求 "WTI" (原油) 数据
**那么** 返回的数据应包含 WTI 原油的历史价格序列

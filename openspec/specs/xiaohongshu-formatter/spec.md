# 小红书格式化工具规范

## 目的

为 xiaohongshu-formatter skill 定义对报告格式美化的行为规范，包括对发散路径板块的识别与格式化。

---

## 新增需求

### 需求: formatter 必须识别并格式化发散路径板块
格式化工具必须能够识别发散路径板块并对其进行 emoji 增强和格式美化。

#### 场景: 识别发散路径板块
- **当** formatter 处理的 `_index.md` 包含 `## 🧭 搜索发散路径` 标题
- **那么** 必须对该板块进行格式化处理
- **并且** 保持表格数据完整性，禁止修改关键词和发散理由的内容

#### 场景: 发散路径格式美化
- **当** 格式化发散路径板块
- **那么** 应当为每轮关键词添加合适的 emoji（根据关键词内容上下文判断）
- **并且** 应当增强发散理由的可读性
- **并且** 必须保持表格对齐格式

#### 场景: 无发散路径时
- **当** `_index.md` 不包含发散路径板块
- **那么** formatter 必须跳过发散路径相关处理
- **并且** 按现有逻辑处理其他板块

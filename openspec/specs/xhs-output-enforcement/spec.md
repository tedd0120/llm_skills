# xhs-output-enforcement 规范

## 目的
定义小红书内容总结输出格式要求，防止 Agent 在执行任务时跳过单篇帖子分析步骤。

## 需求

### 需求: Agent 强制输出流程
Agent 在执行小红书内容搜索和总结时，必须严格遵守以下顺序流程，禁止跳过任何步骤：

1.  **必须创建总目录**：必须在 `data/xiaohongshu/` 下创建一个以主题 and 时间戳命名的独立目录。
2.  **必须生成单篇帖子文章**：对于抓取到的每一篇有效帖子，**必须**在该目录下生成一份独立的 Markdown 文档。
    *   **禁止**将多篇帖子的内容合并到一个文件中。
    *   **禁止**跳过包含单篇主帖摘要和评论区要点的总结环节。
3.  **必须生成汇总索引**：**必须**在所有单篇帖子的 Markdown 文档生成完成后，才生成 `_index.md` 索引文件。
    *   索引文件**必须**包含指向前面生成的每篇独立 Markdown 文件的链接。
    *   **禁止**将索引文件作为唯一的格式化输出文件。

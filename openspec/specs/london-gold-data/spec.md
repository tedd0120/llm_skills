# london-gold-data 规范

## 目的
定义国际黄金（伦敦金/COMEX/现货）行情获取能力的统一规范，确保历史与实时价格抓取逻辑稳定、字段明确，并可直接用于宏观与避险分析。
## 需求
### 需求：伦敦金/国际金价历史数据抓取
系统必须提供国际黄金价格历史数据抓取能力，使用 yfinance 库获取 COMEX 黄金期货或现货金价数据。

#### 场景：抓取COMEX黄金期货历史数据
- **当** 用户请求抓取伦敦金/国际金价，指定日期范围
- **那么** 系统调用 `yf.download("GC=F", start, end)` 抓取 COMEX 黄金期货数据，返回 OHLCV 格式的 DataFrame

#### 场景：抓取现货黄金价格
- **当** 用户请求抓取现货黄金 (XAU/USD) 价格
- **那么** 系统调用 `yf.download("XAUUSD=X", start, end)` 抓取现货金价数据

### 需求：伦敦金实时价格获取
系统必须提供国际黄金实时价格获取能力。

#### 场景：获取COMEX黄金期货实时价格
- **当** 用户请求伦敦金实时行情
- **那么** 系统调用 `yf.Ticker("GC=F").info` 返回当前黄金期货价格信息
